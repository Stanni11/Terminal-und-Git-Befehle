Terminalbefehle

GIT

Was ist Git? - Git ist ein Versionverwaltungstool, das dazu dient dein Projekt abzusichern und Zwischenzuspreichern.

Neue Repository erstellen

  • echo "# 2306" >> README.md
  • git init
  • git add README.md
  • git commit -m "first commit"
  • git branch -M main
  • git remote add origin https://github.com/Stanni11/2306.git
  • git push -u origin main

Neue Repository pushen

  • git remote add origin https://github.com/Stanni11/2306.git
  • git branch -M main
  • git push -u origin main
